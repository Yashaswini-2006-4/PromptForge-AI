# 🚀 PromptForge AI

An AI-powered Chrome Extension Generator that transforms natural language prompts into complete Chrome Extensions using Google Gemini AI.

> Generate • Edit • Save • Download • Deploy

---

## 🌐 Live Demo

**Frontend:** https:prompt-forge-ai-green.vercel.app

**Backend API:** https://promptforge-ai-n5xv.onrender.com

# ⚙️ Installation

## Clone

```bash
git clone https://github.com/Yashaswini-2006-4/PromptForge-AI.git
```

## Backend

```bash
cd backend
npm install
```

Create `.env`

```env
PORT=5000
MONGODB_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET
GEMINI_API_KEY=YOUR_GEMINI_KEY
```

Run

```bash
npm run dev
```

---

## Frontend

```bash
cd frontend
npm install
```

Create `.env`

```env
VITE_API_URL=http://localhost:5000/api
```

Run

```bash
npm run dev
```
